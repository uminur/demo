Responsif desain

pada tampilan website sangat penting untuk diperhatikan. Apalagi di zaman sekarang ini yang apa apa dikerjakan secara online . Maka demi kenyamanan user yang akan menikmati desain website kita, Perlu diperhatikan betul seberapa responsif kah website kita. Berikut beberapa referensi framework CSS yang bisa digunakan dalam membangun website bersifat responsive

1. Bootstrap

Bootstrap merupakan framework css yang paling terkenal diantara framework-framework lainnya dan tentunya paling banyak digunakan. Bootstrap khusus dibangun untuk menjadikan website yang dibuat fully responsive mulai dari tampilan desktop sampai mobile.

Untuk saat ini versi terakhir dari bootstrap adalah versi 3.3.7 rilis tanggal 25 juli 2016 dan versi 4 alpha 3 rilis tanggal 27 juli 2016.

2. PURE

Sekitar tahun 2013 Yahoo! meluncurkan sebuah CSS Module yaitu Pure CSS. Pure dibangun dengan penuh perhitungan, terlihat dari ukuran file css yang kecil. Sehingga ini akan sangat menguntungkan. Versi terakhir rilis pada tahun 2015.

3. Foundation 3

Foundation merupakan framework css website responsif paling canggih yang pernah ada didunia ini, begitulah yang saya kutip dari official websitenya. Apabila Anda ingin membangun web responsif dengan cepat , Maka Foundation adalah pilihan yang tepat, karena mudah digunakan.
